Steps to run symbol-less assembler:

1) Navigate to the directory of the Symbol-less_Assembler.jar file using command prompt.
2) Run the following command (replace FILENAME with the directory and filename of the asm file):
java -jar Symbol-less_Assembler.jar "FILENAME.asm"

Steps to run complete assembler:

1) Navigate to the directory of the Complete_Assembler.jar file using command prompt.
2) Run the following command (replace FILENAME with the directory and filename of the asm file):
java -jar Complete_Assembler.jar "FILENAME.asm"
